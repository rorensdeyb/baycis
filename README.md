	# BayCIS
BayCIS (Bay Central Inventory System) is a web-based inventory management system developed for Bay Central Elementary School. It is designed to replace manual inventory processes with a centralized digital platform that improves asset tracking, accuracy, and efficiency.

🎯 Objectives

The system aims to:

Provide a centralized platform for managing school assets, supplies, and equipment
Enable real-time monitoring of inventory status, quantity, and location
Reduce errors and inconsistencies caused by manual data entry
Generate accurate and timely inventory reports
Improve accountability and utilization of school resources
Enhance overall efficiency in inventory management

❗ Problem Statement

The school currently uses manual inventory methods such as logbooks and spreadsheets, which lead to:

Inaccurate and outdated records
Difficulty tracking asset location and status
Frequent human errors
Risk of asset loss and mismanagement
Time-consuming report preparation
Limited access to reliable data for decision-making

⚙️ Features
📋 Asset Registration and Categorization
📍 Real-Time Asset Tracking (status, quantity, location)
👤 User Management with Role-Based Access
📊 Automated Report Generation
🔄 Asset Transaction Tracking (issuance, return, updates)
🌐 Web-Based Access for Authorized Personnel
🧑‍💻 Target Users
School Administration – for decision-making and reporting
Property Custodian / Supply Officer – for managing inventory records
Inventory Staff – for maintaining organized data
Faculty Members – for transparency and accountability
📂 Scope

The system includes:

Inventory data management
Asset monitoring and tracking
User access control
Reporting and documentation
⚠️ Limitations
No mobile application support (web/desktop only)
No integration with procurement or financial systems
No IoT-based tracking features
Limited for internal school use only
🛠️ Technology Stack (Suggested)


Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: PHP
Framework: Laravel
Database: SQLite / MySQL
